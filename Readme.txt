Weather App

A simple weather application that provides real-time weather updates using the OpenWeatherMap API. Users can search for any city to view its current weather conditions.

Features

Displays temperature, humidity, wind speed, and atmospheric pressure.

Provides weather descriptions and matching icons.

Shows day/night indicators using appropriate icons.

User-friendly interface with a responsive design.

Technologies Used

HTML5

CSS3

JavaScript

OpenWeatherMap API

Prerequisites

A modern web browser (e.g., Chrome, Firefox, Edge)

A stable internet connection

Installation and Setup

Clone this repository:

git clone https://github.com/your-username/weather-app.git

Navigate to the project directory:

cd weather-app

Open index.html in your preferred browser.

Configuration

Obtain your API key from OpenWeatherMap.

Replace the placeholder API key in the script section of index.html:

const apiKey = 'YOUR_API_KEY_HERE';

Usage

Enter a city name in the search box.

Click the Search button.

View real-time weather information.

Troubleshooting

Ensure that the API key is valid and correct.

Check for typos in the city name.

Confirm that the device has an active internet connection.
